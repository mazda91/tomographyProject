\section{Define the frame of work}
~~ For any computer project, a frame of work must be define. As this project is all about medical imagery made of disks of various intensities, the idea of creating three main classes defining images came naturally. 
\paragraph{Framework class} 
This class sets up the area where projections will be taken into account. For convenience, a disk has been preferred, the idea being to be able to translate the center and change the radius(e.g to compute only subsections of an image). Plus, it contains a list of images, so that we can handle them easily, whether it is switching between images, adding or removing.
\begin{itemize}
\item \textbf{Attributes}
	\begin{itemize}
	\item \textbf{center} = [Xcenter,Ycenter]
	\item radius
	\item imageList
	\item currentImage
	\end{itemize}
	
\item \textbf{Methods}
	\begin{itemize}
	\item setters and getters for above attirbutes
	\item addition and removal of an image
	\item plot the framework and the current ilage
	\end{itemize}
\end{itemize}
